關於專案

本專案致力於為藍隊、DFIR 與威脅獵捕專業者提供 AI 深造路線圖與精選資源。聚焦可落地、可度量、可維護。

貢獻指南

  1. Fork 此專案並建立分支。
  2. 新增或修正 assets/data/*.json 或頁面內容。
  3. 提交 Pull Request，說明動機與變更影響。

部署與限制（GitHub Pages）

  • 靜態託管：僅支援 HTML/CSS/JS 與圖片；不支援 PHP/資料庫。
  • 動態資料：透過 assets/data/*.json 客戶端渲染，或以 GitHub Actions 預先產生。
  • 請勿提交含機密資訊的設定；外部 API 請在 CI 取用並提交處理後的 JSON。

Data Schema

resources.json: [
  {{ "title": "...", "desc": "...", "link": "...", "tags": ["ML","DFIR"] }}
]
trends.json: [
  {{ "date": "2025-10-01", "title": "...", "summary": "...", "link": "...", "category": "藍隊" }}
]

Roadmap

  • 自動化 RSS 爬取與去重，生成 trends.json
  • 加入指標面板：precision/recall、MTTD/MTTR、假警報率
  • RAG 驗證清單與資料品質評級
